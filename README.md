# 2. 本地运行

## 1. 使用 yarn（推荐）

先安装`yarn`：

```powershell
npm install -g yarn
```

克隆到本地：

```powershell
git clone git@github.com:Ali-Front-End-Group/study-chinese.git

cd cloud-homework
```

安装依赖包：

```powershell
yarn
```

本地启动项目：

```powershell
yarn start
```

## 2. 使用 npm

克隆到本地：

```powershell
git clone git@github.com:Ali-Front-End-Group/study-chinese.git

cd cloud-homework
```

安装依赖包：

```powershell
npm i
```

本地启动项目：

```powershell
npm run start
```
